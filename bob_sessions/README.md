# IBM Bob Session Evidence

This folder is the required evidence location for the IBM Bob Hackathon submission.

Add the real IBM Bob IDE artifacts here before final submission:

- Exported IBM Bob IDE task history or task session report markdown files
- Screenshots of Bob task session consumption summaries
- Screenshots showing IBM Bob IDE with the relevant mode or session, if available
- Screenshots showing Bob IDE using the `ci-triage` workflow and local MCP tools, if available

Do not add fabricated session exports or fake screenshots. This folder should contain only real evidence captured from IBM Bob IDE or Bob Shell during the project workflow.

Before committing anything in this folder, verify that the files do not contain:

- Credentials, API keys, tokens, or secret values
- Personal data or private account details
- Client data, confidential data, private URLs, or internal organization names
- `.env` file contents or local tool settings

Suggested filenames:

- `bob-session-ci-triage-YYYY-MM-DD.md`
- `bob-task-consumption-summary-YYYY-MM-DD.png`
- `bob-ide-ci-triage-mode-YYYY-MM-DD.png`
- `bob-ide-mcp-analysis-YYYY-MM-DD.png`

If evidence is not yet available, leave this README and `.gitkeep` in place and keep the submission checklist unchecked.
